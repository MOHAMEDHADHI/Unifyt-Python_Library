## üí° New Use Cases Enabled

### 1. Quantum Mechanics
```python
from unifyt import Quantity, constants

# Compton scattering
wavelength_shift = constants.lambda_C * (1 - np.cos(angle))

# Bohr model
radius_n = constants.a_0 * n**2
energy_n = -constants.R_inf * constants.h * constants.c / n**2
```

### 2. Astrophysics
```python
# Schwarzschild radius
r_s = 2 * constants.G * mass / (constants.c ** 2)

# Hubble time
t_H = 1 / constants.H_0

# Luminosity distance
d_L = Quantity(10, 'megaparsec')
```

### 3. Nuclear Physics
```python
# Mass-energy equivalence
mass_defect = Quantity(0.001, 'atomic_mass_unit')
binding_energy = mass_defect * constants.c ** 2
print(binding_energy.to('megaelectronvolt'))
```

### 4. Electrical Engineering
```python
# RC circuit
R = Quantity(1000, 'ohm')
C = Quantity(100, 'microfarad')
tau = R * C
print(tau.to('millisecond'))

# LC circuit
L = Quantity(10, 'millihenry')
C = Quantity(100, 'microfarad')
f = 1 / (2 * np.pi * utils.sqrt(L * C))
```

### 5. Magnetic Systems
```python
# Magnetic field
B = Quantity(1, 'tesla')
print(B.to('gauss'))  # 10,000 gauss

# Magnetic flux
Phi = B * Quantity(1, 'meter^2')
print(Phi.to('weber'))
```

### 6. Radioactivity
```python
# Activity
activity = Quantity(1, 'curie')
print(activity.to('becquerel'))  # 3.7√ó10^10 Bq

# Dose
dose = Quantity(1, 'sievert')
print(dose.to('rem'))  # 100 rem
```

### 7. Data Science
```python
# Data transfer
data = Quantity(1, 'terabyte')
speed = Quantity(100, 'megabyte/second')
time = data / speed
print(time.to('hour'))

# Storage
storage = Quantity(1, 'tebibyte')
print(storage.to('gigabyte'))  # 1099.51 GB
```

### 8. Fluid Dynamics
```python
# Viscosity

print(visc.to('pascal_second'))

# Flow rate
```
